# Phase-Associative Memory — arXiv manuscript

**Repository:** [gowrav-vishwakarma/qllm2](https://github.com/gowrav-vishwakarma/qllm2)

## Build

From this directory:

**Tectonic** (recommended on a fresh machine; downloads TeX packages on first run):

```bash
brew install tectonic
make pdf-tectonic
# or: tectonic main.tex
```

**Traditional TeX Live / MacTeX:**

```bash
latexmk -pdf main.tex
# or
make pdf
```

Requires either [Tectonic](https://tectonic-typesetting.github.io/) or **TeX Live** (MacTeX / BasicTeX) with `latexmk`, `pdflatex`, `bibtex`, and standard packages (`natbib`, `cleveref`, `tikz`, `subcaption`, `booktabs`, `microtype`, `hyperref`).

## arXiv submission

1. Run `make pdf` and confirm `main.pdf` builds without errors.
2. Create a **tarball** containing at minimum:
   - `main.tex`
   - `references.bib`
   - `figures/val_ppl_curve.pdf`
   - `figures/training_run_medium_pam_v3.png`
   - `latexmkrc` (optional)
3. Suggested **primary category**: `cs.CL` (computation and language) or `cs.LG` (machine learning).
4. See [arXiv submission guidelines](https://arxiv.org/help/submit).
